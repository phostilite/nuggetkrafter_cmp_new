Frequently Asked Questions

How to upload a SCORM package?

To upload a SCORM package, navigate to the Upload section, select the file, and click the upload button. Ensure the file is in the correct format.

How to assign a package to a client?

Go to the Client Management section, select the client, and choose the package you want to assign. Click the assign button to complete the process.

How to track user progress?

User progress can be tracked from the User Management section. Select a user to view detailed progress reports and activity logs.
